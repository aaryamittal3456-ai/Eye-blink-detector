# Eye-blink-detector
A real-time Eye Blink Detection Web Application that monitors blinking activity using a webcam to help users maintain healthy screen habits and reduce eye strain.

Features
🎥 Real-time Eye Detection
Uses webcam to monitor face and eye movements
👁️ Blink Tracking
Counts total number of blinks
Calculates blinks per minute
📊 Live Metrics Dashboard
Total blinks
Blink rate (per minute)
Eye Aspect Ratio (EAR)
Session timer
❤️ Eye Health Status
Indicates whether blinking rate is healthy or low
📈 Blink Rate Graph
Visual representation of blink activity over time
🎛️ Controls
Start / Stop / Reset session
Toggle sound alerts
Dark mode UI
📑 Reports Section
View session summaries and analytics
